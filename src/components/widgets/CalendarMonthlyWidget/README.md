# Monthly Calendar Widget

A simple, interactive calendar widget that displays the current month, highlights today, and allows navigation between months.

## Features

- **Monthly View**: See the entire month at a glance.
- **Navigation**: Navigate to previous and next months using chevron buttons.
- **Today Highlight**: The current day is clearly highlighted with a distinct background and bold text.
- **Configurable Start of Week**: Choose which day your week starts on (Sunday, Monday, etc.).
- **"Today" Button**: Quickly jump back to the current month if you've navigated away.
- **Responsive**: Adapts to different widget sizes on your dashboard.

## Usage

Once added to your dashboard, you can:
1. Use the left/right arrows to change the displayed month.
2. Click the settings icon to change the title or the start of the week.
3. Click "Today" to return to the current month.

## Settings

- **Widget Title**: Customize the display name of the widget.
- **Start of Week**: Configure whether your calendar starts on Sunday, Monday, or any other day of the week.

## Responsive Sizing

- **2x2 (Minimum)**: Compact view showing the full month grid.
- **Larger Sizes**: Displays more spacious layout for the calendar grid.
