# Pomodoro Widget

A pomodoro timer widget for time management using focus intervals and breaks.

## Features

- Focus time, short breaks, and long breaks
- Visual indication of current mode with color coding
- Pause, resume, and reset controls
- Cycle tracking
- Audio notification when timer completes
- Responsive layouts for different widget sizes

## Responsive Design

- **2x2**: Minimal view with timer and basic controls
- **4x2+**: Horizontal layout with timer and controls
- **2x4+**: Vertical layout with more statistics
- **4x4+**: Full featured view with all controls and statistics

## Settings

- **Title**: Widget header text
- **Focus Time Duration**: Length of focused work periods (default: 25 min)
- **Short Break Duration**: Length of short breaks (default: 5 min)
- **Long Break Duration**: Length of long breaks (default: 15 min)
- **Cycles Before Long Break**: Focus cycles before a long break (default: 4)
