# Readwise Widget

Displays random highlights from your Readwise account.

## Features

- Display random highlights from Readwise
- Automatic or manual refresh
- View highlight text, notes, book information, and tags
- Configurable refresh intervals
- Responsive layout adapting to widget size

## Setup

1. Get your API token from [Readwise](https://readwise.io/access_token)
2. Add the widget to your dashboard
3. Configure with your API token in settings

## Settings

- **Title**: Widget header text
- **API Token**: Your Readwise API token
- **Refresh Interval**: How often to display a new highlight (minutes)
- **Show Book Information**: Toggle book title and author display
- **Show Tags**: Toggle tags display

## Responsive Design

- **2x2**: Highlight text only
- **3x2**: Adds book title
- **2x3**: Adds author information
- **3x3**: Includes notes and tags
- **4x4+**: Full-featured view with all details
