# Todo Widget

A minimalist to-do list widget for organizing tasks.

## Features

- Add, complete, and delete tasks
- Drag-and-drop reordering
- Customizable sorting (creation date, alphabetical, completion status, manual)
- Option to show or hide completed tasks
- Responsive layout that adapts to different widget sizes

## Usage

1. Add tasks by typing in the input field and pressing Enter
2. Mark tasks as complete/incomplete by clicking the circle button
3. Delete tasks by hovering and clicking the trash icon
4. Drag tasks to reorder (when using manual sort)
5. Access settings via the settings icon in the header

## Settings

- **Title**: Widget header text
- **Sort Order**: How tasks are sorted
- **Show Completed Items**: Toggle visibility of completed tasks
